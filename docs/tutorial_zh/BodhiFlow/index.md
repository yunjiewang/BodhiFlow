# Tutorial: BodhiFlow

BodhiFlow 是一个能将各种*原始内容*（例如YouTube视频、播客或本地文件）自动转换成**结构化智慧文档**的强大工具。
它通过一个两阶段的*流水线*，首先获取并转录内容的原始文字，然后利用多种AI大模型，根据你的需求将文字精炼成总结、纪要或学习笔记等多种格式。所有这一切都可以通过一个直观的图形界面轻松完成。
